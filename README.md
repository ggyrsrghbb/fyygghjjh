第一次看到郑兰生的名字，是在1929年的《文中校刊》上。这本校刊是我在1975年发现的。那时文中毕业后我在家务农，一次我拿一角钱到大队供销社买盐，突然发现装裹货纸的篮子里，有一本旧书《文中校刊》。我眼睛一亮，盐没有买，却用仅有的一角钱买了这本书。

想不到，我保存的这本《校刊》，20多年后1998年，竟成了我写第一本《文中校史》唯一能找到的，上世纪二十年代珍贵史料。1929年的《文中校刊》有学生写的论文、散文、诗歌、小说，最后一节是“会议纪要”。会议纪要记述了校董会汇报詹行烍、陆兴焕下南洋募款概况，主持人是教务主任郑兰生。

后来，我查考文中历年材料，阅读琼海市史志办有关记述，采访郑兰生的子孙后代，终于理清了校长郑兰生不同凡响的生平与教育业绩。 

郑兰生，1892年出生，文昌市龙楼镇光大村人，1916年毕业于国立广东高等师范国文部，历任文昌县教育局局长、琼海中学（今海中）训育主任、文昌中学校长、定安中学校长，1938年秋到广东省立琼崖中学（今嘉积中学）任教务主任，1941年秋任校长。嘉积中学称郑兰生是民国时期嘉中的历任校长中，“任职时间最长，办学成绩最突出的”校长。

四处奔走，北上筹款建设新文中

1936年，文昌中学从蔚文书院原址迁至校场坡。这是文中发展史上具有里程碑式的大事。这次划时代的关键性大搬迁，1928年至1934年三任校长，功标青史。他们历经数年，四处奔走，前赴后继，殚精竭虑，终于建起当时堪称一流的文中新校。许多人都知道，1928年任文中校长的詹行烍率先赴南洋募款，1929年继任的第14届校长陆兴焕下南洋催款，后来还有一位叫陈汝贤的教务主任第三次下南洋。人们把这一时期校长筹款建校的感人经历，称为“三下南洋，十万光洋”。事实上，还有一次重要的校长出访募捐活动，鲜为人知。那就是1934年第17届校长郑兰生北上大陆。他所到之处，受到国内文昌籍同胞的热情接待，踊跃响应。所受捐款，建设校舍面积达2000多平方米。文中现存的仿古建筑“文焕章图书馆”，就是当时郑兰生到上海募捐的。

1934年，文中校场坡新校大部分设施基本完成，部分学生已搬进新校园。但是要建设一个设备完整、功能齐全的一流新文中，仍然需要大笔资金。怎么办？这一年刚任职的校长郑兰生，受校董会委托，决定转向北上，到香港、到南京、到上海，拜访旅居各地的文昌乡亲，募捐筹款建校。他相信，国内乡亲也会像南洋侨胞一样，情怀乡土心系教育。

郑兰生出访的第一站是香港。他孤身一人从海口乘客轮赴港，得到香港的文昌乡亲热烈欢迎。港商邢兰亭、王碧川，著名慈善家周雨亭儿子周文治热情接待这位来自家乡的教育使者。他们纷纷表示答应捐款，资助文中顺利迁往新校。

港事具有端倪，郑兰生风尘仆仆启程南京。那时，广东到南京要坐汽车，搭乘老旧的火车，郑兰生一路奔波，历经四天四夜才到南京城。郑兰生是文昌历史上著名师范院校毕业的第一批优秀教师，曾任文昌县教育局长，知识渊博声名显赫。听说家乡的文中校长来了！南京的文昌籍军政大员、知名人士互相转告，掀起“郑兰生旋风”。陈策、郑介民、郑挺峰、文芗铭、罗华山、黄绩煦、詹守素、郑蔚文、林少波等人，纷纷邀请郑兰生聚餐座谈，听取他介绍文中建校事宜及困难情况，并踊跃认捐校舍，或捐助资金。这次南京之行，收获颇丰。郑介民、郑挺峰合捐学生宿舍一间，文芗铭捐建学生宿舍一间，陈策捐建以其父亲命名的“陈晓山体育馆”一座。

在上海，郑兰生拜会了文焕章的儿子文鸿恩。文鸿恩的父亲是文中教师，因组织学生反对袁世凯复辟，被反动县长杀死。文鸿恩欣然答应捐助4000光洋，兴建以父亲命名的“文焕章图书馆”。

如今，文中校园南面唯一的一间仿古琉璃瓦的标志性建筑，就是当年郑兰生赴上海发动捐建的。

坚持办学  战火中校旗不倒

在郑兰生的教育生涯中，文中是第一站，他的第二站更加壮丽恢弘，精彩夺目！在琼海嘉积中学，人们对郑兰生校长至今念念不忘，他们说，“民国时期，论治校办学，成绩最突出，当推郑兰生”，“嘉中，永远不会忘记郑兰生先生”！

抗日战争时期，郑兰生受任于危难之中，带领几百学生风餐露宿，辗转南北，横渡海峡，创造了海南人抗战中校旗不倒的教育神话，为特殊时期的海南教育谱写了史诗般的的壮丽篇章！

1939年2月，日本侵琼。

侵略军铁蹄所到之处，人们惊慌失措，四处躲避，一时间“书声不再，弦歌声竭”，海南绝大多数的学校停办了。城市容不了，琼崖中学（今嘉积中学）就把书桌搬到山区，搬到日本人汽车通不到的地方。人们不会忘记1939年2月，这一年日军占领了海口，琼东一带实行“焦土政策”，琼崖中学学生忍痛将校园付之一炬，抬着桌椅，从嘉积溪仔街码头上船，沿万泉河溯流而上，转移到到定安的山区蒙养、石壁一带，借用两座祠堂上课。此后，琼崖学生在颠沛流离中，开始了在战火中顽强读书的艰难历程。

几个月后，日军侵占定安县。飞机、炮弹、机枪的声响越来越近，琼崖中学的课桌真的放不下去了。于是，师生们只好向东北迁移。

那是海南历史上唯一的一幅逃亡到大陆办学的壮丽画卷。在这支长长的队伍中，顽强的琼崖师生衣服褴褛，风尘仆仆，进行艰难卓越的教育长征。白天他们躲在山中采野果充饥，晚上披星光伴月影，在荒无人烟的小路上行进。他们三五成群手拉着手，不管多难也不掉队。战火吓不了矢志求学的琼州学子，各县停办学校的学生，纷纷加入这所流亡学校，学校改名为“琼崖联合中学”，实际上就是当时的海南中学。时师生们冒着风险渡过汹涌澎湃的琼州海峡，到达雷州半岛的瑞溪县麻章墟，1939年10月，学校正式上课。

这段时间的校长就是郑兰生。郑兰生一介书生，家庭殷实衣食无忧，他完全可以随着父兄到当时的法租界西营避难，但为了传承琼州教育火种，赓续中华文脉，他勇挑重任不负众望，选择了在血和火中转移迁徙，一路弦歌。在远离家乡的广东山区，他和学生一起砍树割茅搭盖草寮建教室，到田里采野菜捉田蟹裹腹充饥，晚上睡在用竹片搭成的床上，蚊叮虫咬难以入眠。日军飞机来了，他带领学生躲进防空洞，飞机走了学生们又回到教室继续上课，朗朗书声坚毅而顽强。

琼崖联合中学的校旗在粤西的山区冉冉升起，这面在战争中不倒的校旗，飘动着琼州学生英勇不屈的民族骨气，创造了海南教育史上可歌可泣的奇迹！

抗日救国，读书救国！即使极其困难，郑兰生和他的学生们依然严教勤学，毫不放松。课余大家在一起唱《黄河大合唱》《松花江上》。同学们远离海南岛，“迷茫茫不见年老高堂，路漫漫难归琼州吾乡”，不少同学唱到“我的家在东北的松花江上，那里有我的同胞，还有那衰老的爹娘”时，不禁触动情怀，泪流满面。校长郑兰生看到同学们思乡心切，稳步走上土台慷慨陈词：“国难当头，前方将士浴血奋战，我辈诸生当奋志自强，学好本领，为民族光！相信我们一定会胜利，到那时一起漫卷诗书好还乡！”

1942年，春节刚过湛江就沦落入日军之手。地处雷州半岛的琼崖中学，不得不进行第二次大搬迁。据郑兰生回忆录云，“敌陷雷阳，本校乃做第二次大搬迁，余又得与诸生跋涉数百千里之长路，历尽饥寒，备尝险阻，始达化县之宝墟再择地而复校焉”。经多方联系，最后定在宝墟陈始祖祠复课。

身居异乡，校无定迹。学校外日军步步逼近炮火连天，学校里供不应求挨饥受冻，在这种险象处处的处境中，郑兰生从不退缩，绝不轻言放弃。5年的时间，从琼州到雷州，从雷州到化县，从化县到粤北的曲江，最后流亡到山城连县，信宜一带，郑兰生在路上办学，在炮火中兴教，他一直在坚持、坚持，再坚持！

郑兰生带领的这所学校，是当时“广东省唯一坚持办学的重点中学”，“在迁移动荡之中，惨淡经营，规模渐俱，学风日纯”，“在万花纷谢之时，在岭南一枝独秀”。“一时间，两广俊彦负笈从游者，有如风起云涌”。这所学校毕业的学生，有些升入西南联大，有些服务于国家与社会，大不乏人。（郑兰生话）

抗战胜利了。琼崖中学的学生在郑兰生校长带领下，“漫卷诗书喜若狂”，以青春作伴的万丈豪情回到嘉积镇。回来了！嘉积的市民自觉燃起长长的鞭炮，英雄般欢迎战火中坚持办学的郑校长。为了琼崖中学校旗不倒，郑兰生历尽磨难，付出太多太多了！出去时，他身体壮实年富力强，几年后再回来已是满头白发，骨瘦如柴了。

躲避战乱的学生回来了，到哪里上课？郑兰生看中了如今的嘉中校址。这个地方原先是日本人的军营，但此时是国民党188师接管，怎么办？又是这位郑兰生，在关键时刻锚定了嘉积中学的历史。郑兰生多次到广州活动，他向当时在广东任要职的文昌老乡求助，跟教育厅的官员数次斡旋，找人求人拜人，终于将省政府的批文拿到手，1946年冬军队撤出，师生们全部搬入新址，也就是如今的嘉积中学所在地。琼海市原党史办主任陈锦爱说，今天的学生能在嘉中现址读书，“跟民国时期嘉中的一位校长郑兰生有莫大的关系”。

时间过去了80年，那个年代以及后来的风风雨雨，送走了郑兰生。1979年郑兰生去世，享年88岁。他跟普通人一样，静静的躺在龙楼光大村的一片山岗上。“一抔黄土青草压”，无人知道这里掩埋的是在战火中坚持五年的琼崖抗战校长！

郑兰生留下什么？

他带领学生走过风雨，历尽磨难，跨海北上坚持办学，延续了琼崖中学教育文脉，留下了一段壮丽的教育史话；他不惧险阻，冒着危险，在血和火中追求悲壮的梦想，留下了不屈不挠的民族精神。他奉着一颗心，以非凡的毅力和勇气，在极其艰难的岁月，保持琼崖学校校旗不倒，留下了一种精神，一腔骨气，一曲气壮山河的弦歌，一个个凄美的历史故事。

作者：吴亚利

文昌市头苑人。1962—1965年就读文昌中学高中部，毕业后回家务农，后担任民办教师。恢复高考后考取海南教育学院中文专业，1983年分配入文昌中学任教，获中学语文科高级教师职称，先后曾任文昌中学、海口实验中学等校副校长，海口市政协委员，《文昌中学校史》、《百年文中校史》主笔。现任文昌文化研究会副会长，任该会会刊《文昌文化》常务副主编；两宋研究会会刊《守望》常务副主编。

征稿启事
